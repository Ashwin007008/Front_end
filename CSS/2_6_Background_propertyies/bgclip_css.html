Background properties in CSS

This is a paragraph to demonstrate various background properties in CSS. The background color is set to lightblue, and a background image is applied from a URL. The image does not repeat and is centered within the element. The background size is set to cover the entire area, and the attachment is fixed, meaning it does not scroll with the content. The background origin is set to padding-box, and the background clip is set to border-box.
